# de-time-bench

Companion benchmark and reproduction repository for [De-Time](https://github.com/systems-mechanobiology/De-Time).

This repository is the home for benchmark scenarios, leaderboard artifacts, aggregate heatmaps, benchmark-derived methods, and benchmark-paper reproduction assets that are intentionally out of scope for the main de-time package.

## Relationship to other repositories

- Main software package: [systems-mechanobiology/De-Time](https://github.com/systems-mechanobiology/De-Time)
- Benchmark and reproduction assets: systems-mechanobiology/de-time-bench
- Downstream Dec-SR integration consumes stable detime outputs rather than re-embedding benchmark logic inside the package repository.
